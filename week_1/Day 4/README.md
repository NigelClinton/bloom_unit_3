# mymodule

Module that implements Car and Eletric Car classes.

## Usage

```python
from mymodule import Car

# returns 'Ford'
ford.make('Ford')

# returns 'model'
ford.model('Raptor')

# returns '2020'
ford.year('2021')

# returns '0'
ford.milage()

# returns '...Driving'
ford.drive()

# returns '100'
ford.mileage(100)

# returns '...filling up on gas'
ford.get_gas()
```
